## 画面遷移

```
SCR-01 ダッシュボード
 ├─→ SCR-02 演習設定 ─→ SCR-03 問題演習 ─→ SCR-04 結果・解説 ★
 │                          ↑     │ （1問1答は 03⇄04 を反復）
 │                          └─────┘
 ├─→ SCR-05 ブックマーク ─→ SCR-03（再演習）
 ├─→ SCR-06 進捗分析
 └─→ SCR-07 管理
```

画面詳細は [ui/specification.md](../ui/specification.md) を参照。

## 演習モードと画面の対応

| モード | 出題 | 図の表示（FR-17） | 結果表示 |
|--------|------|------------------|---------|
| ランダム1問1答（FR-02） | 1問ずつ | 設定に従う | 1問ごとに SCR-04 |
| 分野別ミニテスト（FR-01） | 10/20問 | 設定に従う | 全問回答後に SCR-04（一覧＋各問解説） |
| 模擬試験（FR-04） | 65問/130分 | 強制的に「解説時のみ」 | 採点後に SCR-04（合否＋分野別内訳） |
